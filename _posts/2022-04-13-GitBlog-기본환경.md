1. ruby 설치
   - [Ruby 다운로드 Site](https://www.ruby-lang.org/en/downloads/)
1. cmd 에서 bundler 설치
   - ```
     gem install bundle
     ```
1. 해당 블로그 디렉토리에서 bundle 설치

   - ```
     cd /MyblogDir
     bundle install
     ```

1. webric 추가
   - ```
     bundle add webrick
     ```
1. local로 띄우기

   - ```
     bundle exec jekyll serve
     ```
